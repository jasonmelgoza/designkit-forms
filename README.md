# Designkit Forms

## Install

```bash
npm i designkit-forms
```

## Usage

```html
<form>
  <label for="my_input">Form Label</label>
  <input id="my_input" class="form-control" type="text" name="name" value="" placeholder="Placeholder text...">
</form>
```

## The CSS

//
// Designkit-Buttons
// --------------------------------------------------
*/

  margin: 0;
  border: 0;

  font-size: 11px;

  margin-bottom: 0.3rem;
